\section{Функции и графики}

При построении графиков необходимо отмечать на графике
координаты точек пересечения с осями координат, точки экстремума,
точки перегиба, асимптотику на бесконечности
и вблизи особых точек, области выпуклости графика.

\subsection{Предварительная подготовка}

Построить графики элементарных функций: $f(x) = \hm{x}$, $f(x) = x^2$, $f(x) = \sqrt x$.

\subsection{Преобразования графиков элементарных функций}

Постройте график функции

\begin{nums}{-2}
\item $f(x) = \bm{\hm{x-1}-\hm{x+2}}$.
\item $f(x) = \hm{(2x+1)^2 - 9}$.
\item $f(x) = \frac{(x-1)(x-2)}{2x}$.
\item $f(x) = (2x-1)^3+\frac12$.
\item $f(x) = \frac{2x+3}{3x-4}$.
\end{nums}

\subsection{Более сложные примеры}

\begin{nums}{-2}
\item $f(x) = x^4 -2x^3+3x^2-4x+5$.
\item $f(x) = x^2 - \frac{5}{\sqrt x}$.
\item $f(x) = \frac{x^3 - 2x^2 + 4x -1}{x^2+2x-3}$.
\item $f(x) = \frac{(x-2)(x^2-4x+3)(x-3)}{x^2-5x+6}$.
\item $f(x) = \frac{1}{1 + \sqrt{x+5}}$.
\item $f(x) = \frac{1}{e^x-1}$.
\item $f(x) = e^{-x^2}$.
\item $f(x) = \log_2 (x+1)$.
\item $*$ $f(x) = \frac{\log_2 x}{x+1}$.
\item $*$ $f(x) = x\log_2 x$.
\end{nums}
