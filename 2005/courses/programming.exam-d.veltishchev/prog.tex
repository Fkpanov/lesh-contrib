\section*{Вариант 1}

\textbf{1.} Идентификаторы (какими они могут быть?). Объявление переменных, констант.
Стандартные типы данных \Cpp и их назначение. Примеры. Массивы: объявление, обращение к элементам массива,
особенности индексации (нумерации элементов).

\textbf{2.} Найдите ошибку в программе:

{\footnotesize\hskip0.7cm\vbox{
\begin{verbatim}
int main()
{
    int n = 100;
    int arr[n];
    for (int i = 0; i < n; i++) arr[i] = i;
    return 0;
}
\end{verbatim}}}

\textbf{3.} \textit{Пузырьковая сортировка:} написать программу для сортировки по \textbf{возрастанию} массива из 1000 элементов, вводимого с
клавиатуры с помощью оператора \ttt{scanf}. \emph{Метод сортировки:} последовательно сравниваются все пары
\textbf{соседних} элементов с номерами \ttt{i}, и \ttt{i+1}, если \ttt{a[i]>a[i+1]}, то переставляем эту пару
местами (здесь \ttt{i} пробегает все значения от 0 до 999). Когда доходим до конца массива, начинаем сначала, и так далее.
Процесс сортировки нужно прекращать, если при проходе по всему массиву от начала до конца не осуществлено ни одной
перестановки (то есть все элементы уже упорядочены и нет ни одной <<неправильной>> пары.

\textbf{3.} Дана строка символов (\ttt{char * s1 = "..."}) некоторой неизвестной заранее длины, и подстрока
\ttt{char * s2} м\'еньшей длины. Требуется найти количество вхождений подстроки \ttt{s2} в строку \ttt{s1}.
Вхождения могут перекрываться (например, подстрока \ttt{"aaa"} входит в строку \ttt{"aaaaa"} три раза).

\section*{Вариант 3}

\textbf{1.} Условный оператор \ttt{if}. Оператор выбора \ttt{switch}.
Объявление процедур и функций. Функция \ttt{main()}. Оператор \texttt{return}.

\textbf{2.} Найти ошибку в программе:

{\footnotesize\hskip0.7cm\vbox{
\begin{verbatim}
int main()
{
    int x[100];
    for (int i = 0; i < 100; i++) x[i] = 100-i;
    int * y = (int*) x;
    int ** z = &y;
    for (int i = 0; i < 100; i++) (**z)[i]++;
    for (int i = 0; i < 100; i++) printf("%d ", y[i]);
    return 0;
}
\end{verbatim}}}

\textbf{3.} Даны два массива \ttt{a1} и \ttt{a2} из целых чисел, имеющие длины \ttt{l1} и \ttt{l2} соответственно. Имеется также третий массив,
длина которого равна \ttt{l1+l2}. Требуется отсортировать оба массива и слить их в третий, затратив на это \ttt{l1+l2} операций
присвоения, и третий массив должен при этом остаться упорядоченным.
