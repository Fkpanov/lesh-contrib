\centerline{\LARGE \bf Занятия по программированию}

\bigskip


Я планирую параллельно организовать две серии занятий. Одна из них будет, скорее всего, повторять (и дополнять)
курс, прочитанный на ЛЭШе этим летом. Она будет называться <<Введение в программирование на C++>>
и будет рассчитана на слушателей с минимальной подготовкой. Вторая\т для более продивнутых\т будет посвящена
алгоритмам и структурам данных.

Приглашаются все желающие, с любых отделений\т математического, физического, биологического и гуманитарного.
Для тех, кто выберет этот курс в качестве основного, будет зачёт, который будет состоять из теоретического вопроса
и задачи (\те реализации и отладки некоторой программы).

\medskip

Ниже приведу примерную программу курсов.

\section*{Введение в C++}

Первые 3 пункта могут быть пропущены при наличии должной подготовки у слушателей.

\begin{items}{-3}
\item Архитектура компьютера.
\item Системы счисления.
\item Алгебра логики.
\item Типы данных C/C++. Переменные.
\item Условные операторы, операторы циклов.
\item Организация хранения данных в памяти. Адреса, указатели, массивы, ссылки.
\item Процедуры, функции.
\item Исполнение программы процессором. Память, регистры.
\item Основы ассемблера.
\item Стек вызова процедур. Передача параметров в стеке.
\item Типы, объявляемые пользователем.
\item Основы объектного программирования: структуры, классы.
\end{items}

\section*{Структуры данных}

Здесь я буду ориентироваться на слушателей, которые достаточно хорошо знают
хотя бы какой\д нибудь язык программирования из следующего набора: C/C++, Java, Pascal/Delphi.

\begin{items}{-3}
\item Массивы как простейшие структуры данных. Сортировки.
\item Теорема о нижней оценке сложности сортировки произвольного массива.
\item Линейные структуры данных: стек, дек и список.
\item Деревья.
\item Множества и hash-функции.
\item Некоторые алгоритмы сжатия данных.
\item Реализация структур данных на C++.
\end{items}
